# public/

Static assets and PWA files served directly by Vite.

## Files
- `manifest.json`: PWA metadata (`name`, `icons`, `start_url`, `display`, colors).
- `outfit.png`: App image used by the manifest and general project branding.
- `placeholder.svg`: Generic placeholder image asset.
- `robots.txt`: Crawler directives (currently allows all major bots and all paths).
- `sw.js`: Service Worker scaffold (install/activate hooks are present, fetch caching logic is minimal).
